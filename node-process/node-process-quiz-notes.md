# node-process-quiz-notes

## Quiz Questions

After completing this exercise, you should be able to discuss or answer the following questions:

- What is a computer process?
  A computer process is an instance of a computer program that is being executed by the operating system. It represents the state of a running program and includes the program's code, data, and resources. A process has its own memory space, registers, and other attributes that allow it to execute independently.

- Roughly how many computer processes are running on your host operating system (Task Manager or Activity Monitor)?
  50+

- Why should a full stack Web developer know that computer processes exist?
  To help optimize applications, manage multiple simultaneuous requests or performing tasks in parallel, configure servers efficiently, deploy and scale applications properly, debugging
